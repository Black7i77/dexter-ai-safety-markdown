Privacy notice

Effective 5 July 2026

Dexter is a small public beta operated from a locally hosted computer. Do not submit passwords, private keys, banking information, identity documents, medical records, or other highly sensitive information.

How chat works

Your message travels through the public web connection to the Dexter server and is sent to an Ollama model running on the host computer. The generated answer is streamed back to your browser.

Conversation history

Conversation history is stored in your browser using local storage so it can appear again on that device. Dexter does not currently provide user accounts or a server-side conversation database. Clearing browser storage or using the Clear History control removes that browser copy.

Technical information

The service may process basic connection information such as an IP address, request time, browser request headers, and error details for security, rate limiting, and troubleshooting. Traffic delivered through Cloudflare is also processed by Cloudflare under its own terms and policies.

Optional current-information verification

The operator may enable a web-verification feature for clearly time-sensitive public questions. When enabled, a shortened version of that public search question is sent from the Dexter server to Ollama's web-search service so Dexter can use current sources. Passwords, verification codes, private keys, banking details, medical records, and similar sensitive prompts are excluded from automatic search. The feature is disabled unless the server operator configures it.

AI limitations

Prompts and responses are processed to provide the requested AI answer. Dexter can make mistakes. Important technical, financial, legal, medical, or safety-related information should be independently verified.

Beta changes

This notice may be updated as Dexter gains accounts, analytics, databases, or other features. Material changes should be reflected on this page before those features are used publicly.
